# PDFGeneratorSample

## Project Overview

`PDFGeneratorSample` is a demonstration project that shows how to generate PDF documents programmatically. The project uses .NET (C#) and integrates with a PDF generation library to create, customize, and export PDF files from various data sources or templates.

## Features

- Generate PDF documents from C# code
- Support for custom text, tables, and images
- Example of exporting data to PDF
- Simple API for PDF generation
- Error handling and logging
- Sample unit tests

## Requirements

- .NET 6.0 or newer
- PDF generation library QuestPDF (https://www.questpdf.com/)
- Visual Studio or VS Code recommended
